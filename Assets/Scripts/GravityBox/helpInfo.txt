Доступные переменные и методы для GravityBox:

- gravityX: double
  Описание: Гравитация по оси X. Допустимые значения: от -5 до 5.
  Пример: gravityX = 2; // Увеличивает гравитацию по оси X.

- gravityY: double
  Описание: Гравитация по оси Y. Допустимые значения: от -5 до 5.
  Пример: gravityY = -1; // Обратная гравитация по оси Y.
